Minimum Distance Visualizer

Find 3 identical numbers in the array. We want them to be as close together as possible.

💡 The Visual Trick:

  • We don't need to check all combinations. We only check consecutive triplets (e.g., the 1st, 2nd, and 3rd appearance of a number).
  • The distance formula |i-j| + |j-k| + |k-i| exactly equals 2 × (Span).
  • The Span is simply the physical gap between the first and last element of the triplet (k - i).
