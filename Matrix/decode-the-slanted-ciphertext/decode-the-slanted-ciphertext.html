How it Works: Problem Explanation

The Cipher: The original text is encoded by placing it into a matrix along downward-sloping diagonals (↘). Once the matrix is filled, the encoded text is generated by reading the matrix row by row (→), from top to bottom.

The Separation Principle (1D String to 2D Grid) How does a single continuous line of text get separated into a grid? Because the text was extracted row-by-row during encoding, we rebuild the grid by splitting the string into equal-sized chunks. First, we calculate the columns: cols = total length / rows.
  • Chunking Concept: The first chunk of characters (from index 0 to cols-1) forms the top row. The second chunk forms the next row, and so on.
  • Mathematical Coordinates: If you look at the raw input string, any character at index i maps exactly to its grid location using basic math:
    Row = Math.floor(i / cols) and Column = i % cols.

1 Form the Matrix

Total cells = text length. cols = length / rows. We lay out the encoded text left-to-right, row-by-row.
